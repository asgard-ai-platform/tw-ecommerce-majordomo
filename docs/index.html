中文裡的「掌櫃」、「總管」是有溫度的詞——
指替頭家管事的可靠人。不是雇員、不是執行者，而是把整間店當成自己事業在跑的內部人。

tw-ecommerce-majordomo 是為台灣電商打造的 agent plugin。 把 29 個 tw-ecom-* skills 與 12 個 MCP servers 一次塞進你的 coding agent—— 金流、物流、發票、合規、客服這些瑣碎卻致命的事，讓它像有十年台灣電商經驗的營運總管。

每一檔店的日常

台灣電商的四個致命陷阱，
majordomo 怎麼接。

痛點

客人說付了錢，但訂單顯示未付款。是 callback 沒驗章、重複觸發、還是 Shopline 沒同步？

majordomo 怎麼接

tw-ecom-payment-ecpay 封裝 CheckMacValue 驗章，配 ecpay MCP 查授權結果，三邊對齊一鍵。

痛點

訂單成立就開發票，退貨潮一來，每天都在作廢、跨期還變折讓單。

majordomo 怎麼接

tw-ecom-invoice-ezpay + tw-ecom-invoice-void 把財政部「出貨後 48 小時內」與跨期作廢/折讓判斷邏輯寫好。

痛點

超商取貨付款 COD 對帳對不上，物流取件成功的、金流入帳的、訂單狀態三邊永遠差幾筆。

majordomo 怎麼接

ecpay + ecpay-logistics MCP 配 tw-ecom-payment-ecpay，三邊比對含 T+N 結算缺口。

痛點

雙 11 前一個月才開始想：選哪些品、滿額怎麼設、廣告分蝦皮還是 LINE OA。

majordomo 怎麼接

tw-ecom-operations-promotion + tw-ecom-analytics-benchmarks 把選品、ROI 預估、檔期通路規則一次跑完。

命名由來

Majordomo 從拉丁文 majordomus（家中之首）來，
原指中世紀貴族府邸或皇室的總管。
對應中文的「掌櫃」、「總管」——替頭家統管全部事務的內部人。
